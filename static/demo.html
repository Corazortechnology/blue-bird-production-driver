Driver Safety — browser demo

Uses the same WebSocket protocol as data_pipeline/client.py (JPEG in → JSON + JPEG out).

You opened this page over plain HTTP on a public IP. Chrome/Safari/Firefox will not expose the camera API here (navigator.mediaDevices is missing) — this is expected.

Quick test (works today): on your Mac run: 
kubectl port-forward svc/api-service 8080:80
Then open http://localhost:8080/demo (camera allowed on localhost).

Share with others: put HTTPS in front (Azure App Gateway, Ingress + cert-manager, Cloudflare Tunnel, etc.) and use https://your-domain/demo.
Camera rules: secure context only — HTTPS or http://localhost. Plain http://20.x.x.x will not get a camera.
